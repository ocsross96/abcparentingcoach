Accessibility

ABCParenting tries to ensure as much as possible that it's media and content is accessible to people who have visual, hearing, motor or cognitive impairments. It does this in the following ways.

  • Supports the enlarging of text. (Except image replacement headings which are already a large font size. Users MUST be able to resize text
  • Uses plain language and avoids jargon.
  • Divides large blocks of information into manageable chunks e.g. uses short paragraphs and use of lists to present information
  • Provides alt text for all editorially significant images.
  • Provides a consistent navigation model.
  • Clearly defines the different sections of the page and ensure consistent location of screen objects.
  • Supports browser back button functionality
